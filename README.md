# SHE Navigation Engine Starter

This is a simple GitHub-ready starter version of the SHE navigation concept.

## What it includes

- Homepage
- Symptom search
- SHE Pathways
- Explore SHE section
- SHE Facts / Finds / News / Asks / Tells / Services placeholder pages

## How to use

1. Upload these files to a GitHub repository.
2. Go to Settings → Pages.
3. Deploy from the main branch.
4. Open the GitHub Pages URL.

## Current demo searches

## Important

This is educational navigation only and does not provide diagnosis or medical advice.
