# SHE Navigation Engine MVP

A GitHub Pages-ready MVP for SHE: a women’s health navigation search engine.

## Files
- `index.html` — page structure
- `style.css` — design and responsive styling
- `script.js` — pathway engine, keyword recogniser and results logic

## How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload these three files to the root of the repo.
3. Go to Settings → Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select `main` and `/root`.
6. Save.

## Safety
This MVP is educational and navigation-focused only. It does not provide diagnosis, treatment or emergency support.
