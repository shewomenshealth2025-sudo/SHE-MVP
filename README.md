# SHE Navigation Engine

A GitHub Pages-ready MVP for SHE: a simple women's health navigation search experience.

## Files
- `index.html`
- `style.css`
- `script.js`

## How to publish
1. Upload all files to a GitHub repository.
2. Go to Settings > Pages.
3. Select your main branch as the source.
4. Open your GitHub Pages URL.
